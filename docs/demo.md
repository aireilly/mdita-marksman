# Demo

MDITA Marksman provides the following language features:

- **Document symbols**: Navigate document structure via headings.
- **Hover preview**: Preview link targets on hover.
- **Completion**: Complete wiki links, inline links, reference links, and YAML metadata keys.
- **Find references**: Find all references to a heading or link definition.
- **Project-wide diagnostics**: Detect broken links, ambiguous references, and MDITA compliance issues.
- **Rename refactor**: Rename headings and reference links across the project.
- **Table of Contents**: Generate and update a table of contents via code action.
